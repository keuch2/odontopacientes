# Assets - Imágenes

Esta carpeta contiene los placeholders de imágenes que serán reemplazados por los assets finales.

## Imágenes necesarias según mockups:

### Logo
- `logo-main.png` - Logo principal de OdontoPacientes
- `logo-white.png` - Logo blanco para fondos oscuros

### Pantalla de Inicio/Landing
- `hero-landing.png` - Imagen hero de la landing page
- `feature-1.png` - Primera característica
- `feature-2.png` - Segunda característica
- `feature-3.png` - Tercera característica

### Ilustraciones
- `empty-state.png` - Estado vacío para listas sin resultados
- `success.png` - Ilustración de éxito
- `error.png` - Ilustración de error

### Avatares
- `avatar-placeholder.png` - Avatar por defecto de usuario
- `patient-placeholder.png` - Avatar por defecto de paciente

## Especificaciones:
- Formato: PNG con transparencia donde sea necesario
- Resoluciones: @1x, @2x, @3x para soporte multi-densidad
- Optimizadas para peso reducido
